# SnowTricks

SnowTricks est un site internet présentant des figures de snow.

### Pré-requis
    * PHP
    * Composer
    * Symfony cli
    * Docker
    * Docker-compose

-> symfony check:requirements

### Lancer environnement de développment

composer install
npm install
npm run build
docker-compose up -d
symfony server:start -d


#### Lancer un test

php bin/phpunit --testdox
